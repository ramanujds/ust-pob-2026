
## Problem 1: Student Grade Analyzer (List + Map)

**Task**: Read student data and compute grade statistics.

**Input**:
```
First line: n (number of students)
Next n lines: name score
```

**Sample**:
```
Input:
5
Alice 92
Bob 78
Charlie 85
David 59
Eve 95
Output:
Total: 5, Avg: 81.8
A:2 B:1 C:1 D:0 F:1
```

## Problem 2: Shopping Cart Duplicate Remover

**Task**: Process shopping cart with duplicate items.


**Sample**:
```
Input:
6
milk
bread
milk
eggs
bread
cheese
Output:
Original: 6, Unique: 4, Duplicates: 2
Sorted unique: bread, cheese, eggs, milk
```

## Problem 3: Word Frequency Counter 

**Task**: Count word frequencies from multiple sentences.


**Sample**:
```
Input:
3
I love java
java is great
love coding
Output:
java:2 love:2 is:1 great:1 coding:1 I:1
Top 3: java(2), love(2), is(1)
```

## Problem 4: Employee Salary Groups 

**Task**: Group employees by salary brackets.

**Sample**:
```
Input:
4
Alice 45000
Bob 75000
Charlie 120000
David 60000
Output:
Low (<50000): Alice
Mid (50000-100000): Bob, David
High (>100000): Charlie
```
